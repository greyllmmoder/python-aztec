# Code of Conduct

## Our standard

This project is committed to a respectful, inclusive, and constructive environment.

## Expected behavior

- Be respectful in language and feedback.
- Focus discussion on technical substance.
- Assume good intent and clarify before escalating.
- Accept and act on constructive review.

## Unacceptable behavior

- Harassment, personal attacks, or discriminatory remarks.
- Trolling, intimidation, or repeated disruptive behavior.
- Sharing private information without consent.

## Enforcement

Maintainers may edit, remove, or reject contributions and comments that violate this code.

## Reporting

Report issues privately to the maintainer via repository security contact channels.
